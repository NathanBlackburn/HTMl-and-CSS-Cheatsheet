HTML and CSS Cheatsheet

HTML Elements

Tag Name Description
<body> Body Tag Any visible content needs to be placed in the opening and closing tags
<h1 to h6> Subheading Tags Used to provide titles for sections of content
<p> Paragraph Specify text or blocks
<span> Span Text Used to seperate small pieces of content that are on the same line as other content
<div> Divison Container that divides the page into sections
<em> Emphesis Tag Em will render the text as italic
<strong> Strong Tag Strong will render the text as bold
<br Line Breaks Modify the space in the browser with br
<ol> Orderd List Make a list ordered in numbers
<ul> Unordered List Make a list unordered
<li> List Items Tag Add items to a list
<img> Image Tag Allows you add an image to the page. slef closing tag. requires an src attribute
<table> table tag creates a table
<tr> Table Row Adds a row to the table
<td> table data enables you to add data to a row
<th> Table Heading Create a table heading
<tbody> Table Body The body of the table
<thead> Table Head The head of the table
<tfooter> Table Footer The footer of the table
<header> Header Tag Container for navigational links and introductory content
<nav> Nav Link Define a block of links
<main> Main Tag Encapsulates the main content in a web page
<footer> Footer Tag Content at the bottom of subject information
<article> Article Tag Holds content that makes sense on its own
<section> Section tag Defines elements in a document with the same theme
<aside> Aside Tag Mark additional information that can enhance another element but isnt required in order to understand the main content
<figure> Figure Tag Used to encapsulate which is refrenced in the main flow of the document
<figcaption> Figcaption Tag Describe the media within the figure tag
<audio> Audio Tag Embed audio content into a document
<video> Video Tag Add videos to the website
<!DOCTYPE html> Document Type Decleration Tells the browser what type of document and version of HTML is being used
<html> Html Tag Anything inbetween these tags is interpreted as HTML
<head> Head Tag Contains the metadata for the webpage
<title> Title Tag The title for the webpage in the browser
<a> Anchor Tag Enables you to add links with the anchor tag
<embed> Embed Tag Incorporate media content into a page

CSS Elements

Tag Name Description
<style> Internal/ External Stylesheet Write CSS within the head of a HTML document
<link> Link element Link HTML and CSS files together. Placed within the head of the HTML document.
<*> Universal Selector Selects all elements of a given type
<p:hover> Hover Pseudo-class Enables a color or anything else when a user hover over an element which :hover is attatched too
<p:focus> Focus Pseudo-class When an element is clicked, a border is added showing it is in focus
<p:visited> Visited Pseudo-class Click on a blue link to visit another page, when you return the link has changed to purple
<p:disabled> Disabled Pseudo-class When you are on a form and haven't filled it all out, the submit button is grey and disabled.
<p:active> Active Pseudo=class When the form has been filled out, the submit button is blue and enabled
<font-family> Font Family Tag Enables you to change the font
<font-size> Font Size Tag Enables you to change the size of your font
font-weight Font Weight Tag Enables you to change the weight of your font, for example bold or normal etc
<text-align> Text Align Tag Enables you to align text using right, left, center and justify
<color> Foreground Color Styles an elements foreground color
<background-color> Background Color Styles an elements background color
<opacity> Opacity Tag Measures how transparent an element is. Measured from 0% to 100%
<background-image> Background Image Property Set the elements background to an image. Must be linked to an image with a URL
<!important> !important Tag Will override any style no matter how specific
<position> Position Property Changes the default position of a property. Takes 1 of 5 values
<static> Static Position The default position for property
<:relative> Relative Position Enables you to position an element relative to its default stattic position on the web page. Requires an offset property of top, bottom, left and right
<:absoulete> Absoulete Positon All other elements on the page will ignore it. The element will be positioned relative to its closest positioned parent element and then offset properties can be used to determine the final position from there
<:fixed> Fixed Position Enables you to fix an element to a position on the page regardless of user scrolling. Used with offset properties top, bottom and left and right
<sticky> Sticky Position Keeps an element in the document flow but sticks to a relative position. With offset properties
<z-index> Z-Index Position Controls how far back or forward an element should appear on the page when elements overlap. Accepts integer values
<:inline> Inline Property Inline elements have a box wrapped around the text, only taking up as much space as necessary
<:block> Block Property Fills the entire width of the page by deafult, width can be modified however
<inline-block> Inline Block element A combination of both inline and block. They can appear next to each other and can their dimensions can be specified with height and width
<> Float Property Used for wrapping text around an image and can move an element as far left or right as possible in the container. Uses left or right
<float> Float Property Specifies how elements should behave when they bump into each other on the page. Takes 4 values: left, right, both and none
<font-style> Font Style Takes different values to change the style of the font
<text-transform> Text Transform Allows you to either change all the fonts to uppercase or lowercase
<letter-spacing> Letter Spacing Allows you to set the space inbetween the letters. Takes px or em
<word-spacing> Word Spacing Allows you to set the space inbetween the words. Takes px or em
<line-height> Line Height Allows you to set the heigt of the lines. Can take unitless numbers, px, em or %

CSS attributes

Tag Name Description
style Styling Tag Added to the opening tag, set it equal to the adress of the CSS file
href N/A Creates a hyperlink. The value must be the adress to the CSS file
rel Relationship Describes the relationship between the HTML file and the CSS file
